Alan Turing



Alan Turing was a european mathematician and he is considered by some to be the father of computer science as he had made many substantial developments in the computer science field. Two things that Alan are most renowned for are his work on Turing machines and the Turing test. The Turing machine is a theroretical machine proposed by Alan Turing which describes a machine that can determine whether any task it has been given is "circular" or an infinte task and if any random task given to the machine will print out a given symbol and he was able to determine both of these cases as false as in no computer would ever complete these tasks. This proved that there are fundemental limitations on mechanical computation. The Turing test was a test propsed by Alan Turing to determine if a machine can show intelligence and behavior similar and almost indistinguishable from a human. This is done by having a human judge look at a conversation between a human and a computer and determine who the computer is and if the judge isn't able to correctly identify the machine with complete certainty than the machine would have passed the test.
